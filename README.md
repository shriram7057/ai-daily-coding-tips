# ⚙️ Architecture

## generate.py

Responsible for:

- Selecting educational topics
- Generating markdown content
- Creating timestamped files
- Organizing content by date

## scheduler.py

Responsible for:

- Preventing excessive automation
- Limiting posting frequency
- Restricting activity to daytime hours
- Encouraging healthy repository patterns

## GitHub Actions

Responsible for:

- Running automation hourly
- Executing scheduler checks
- Publishing new tips
- Automatically committing updates

# 🤖 GitHub Actions Setup

1. Push repository to GitHub
2. Open repository
3. Navigate to:
   Settings → Actions → General
4. Enable:
   - Read and write permissions

GitHub Actions will automatically run every hour.

# 🛠 Troubleshooting

## Workflow Not Running

Check:

- Actions tab enabled
- Workflow permissions enabled
- YAML syntax correctness

## Python Not Found

Ensure Python is added to PATH.

Verify:

```bash
python --version
```

## Git Push Errors

Re-authenticate GitHub credentials.
